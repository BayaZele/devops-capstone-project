# devops-capstone-project

This project is part of a DevOps capstone lab. It focuses on building a customer account microservice using a REST API.

The service will support:
- Create customer accounts
- Read customer details
- Update customer information
- Delete customer accounts
- List all customers

The project also includes DevOps practices such as GitHub project planning, backlog refinement, and sprint planning.
